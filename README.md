# @qgisk/prettier-config

## Install

```npm
npm i @qgisk/prettier-config
```

## Usage

In your package.json add

```json
{
  "prettier": "@company/prettier-config"
}
```

Or if you prefer a .prettierrc

```
"@qgisk/prettier-config"
```
